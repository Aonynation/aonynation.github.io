分析

题目理解不再进行解释。

可以考虑从另外一种角度去思考问题，即正难则反的方法。

把题目的主要研究对象由牛转变为比赛项目。

那么就可以利用类似状态压缩的方法解决问题，定义 dp 数组 \(f_{i,mask}\) 。

其中 \(i\) 表示已经处理到了第 \(i\) 个比赛任务。

而 \(mask\) 则是对当前有那些牛参加任务的一个状态的压缩，参加了为 1 没参加为 0 。

接下来考虑如何转移，由特殊到一般，我们先舍弃另外的奖励活动。

定义 \(bmask\) 为上一级的压缩状态，\(j\) 为当前的牛， 那么会存在： \[ f_{i,mask}=\max\{f_{i,mask}\ ,f_{i-1,bmask} \ + \ val_{i,j}\} \]

在加入一个奖励条件也就是非常简单的贪心问题了。

  • 由感性理解，一定要按照时间进行排序。
  • 如果时间一样，再按照要求的分数限制从小往大排序。

注意需要一个指针 tap 随时记录位置，可以减小复杂度。

至此题目解决，时空复杂度均为 \(O(n2^n)\) ，可以使用滚动数组但没有必要。

code

```cpp #include <bits/stdc++.h>

#define File(a) freopen(a".in", "r", stdin), freopen(a".out", "w", stdout)

const int N = 21;

int n, m, val[N][N], f[N][1 << N], ans, tap;

struct Node { int t, lim, val; } node[N]; bool operator<(const Node &p, const Node &q) {return (p.t == q.t) ? (p.lim < q.lim) : (p.t < q.t);}

template inline void read(T &a);

signed main(void) { // File("1"); read(n), read(m); for (int i = 1; i <= m; i ++) read(node[i].t), read(node[i].lim), read(node[i].val); std::sort(node + 1, node + 1 + m); for (int i = 1; i <= n; i ++) for (int j = 1; j <= n; j ++) read(val[j][i]); tap = 1; for (int i = 1; i <= n; i ++) { while (node[tap].t < i && tap <= n) tap ++; for (int j = 0; j <= (1 << n + 1) - 1; j ++) { if (__builtin_popcount(j) != i - 1) continue; for (int k = 1; k <= n; k ++) { int now = k - 1; if (j & (1 << now)) continue; int mask = j | (1 << now), res = f[i - 1][j] + val[i][k]; for (int ip = tap; ip <= m && node[ip].t == i; ip++) res += (res >= node[ip].lim) * node[ip].val; f[i][mask] = std::max(f[i][mask], res); } } } for (int i = 1; i <= (1 << n + 1) - 1; i ++) ans = std::max(ans, f[n][i]); std::cout << ans << std::endl; return 0; }

template inline void read(T &a) { T s = 0, t = 1; char c = getchar(); while (!isdigit(c) && c != '-') c = getchar(); if (c == '-') c = getchar(), t = -1; while (isdigit(c)) s = s * 10 + c - 48, c = getchar(); a = s * t; }
